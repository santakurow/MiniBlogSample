# MiniBlogSample

Rails6の新機能であるActionTextを使った簡単な記事投稿アプリを実装。  
画像をアップロードする際の拡張子と画像容量サイズの制限も設けている。  
フロントはBootstrapを使用。  


#### 言語

* Ruby 2.6.4
* Haml
* Javascript

#### フレームワーク

* Ruby on Rails 6.0.3

#### ライブラリ

* jQuery 3.4.1
* Bootstrap

#### DB

* MySQL

#### 使用技術・ツール

* ActionText
* ActiveStorage
* haml-rails (gem)
* image-processing (gem)
* webpack
